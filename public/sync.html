CLI Sync

You can also export/import from the command line using the reporter tool:

# Export all agents to a file
npx session-forge-reporter sync --hub http://your-hub:3700 --key sfh_admin_... --output backup.json

# Export specific agents
npx session-forge-reporter sync --hub http://your-hub:3700 --key sfh_admin_... --agents agent1-id,agent2-id --output backup.json

# Import from a backup
npx session-forge-reporter sync --hub http://your-hub:3700 --key sfh_admin_... --import backup.json
