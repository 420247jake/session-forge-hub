Terms & Privacy

The short version: your data is yours, we collect nothing

TL;DR

  • session-forge hub is free, open-source software (MIT license).
  • It runs entirely on your server. We have no access to it.
  • We collect zero data. No analytics, no telemetry, no phone-home.
  • Your data never leaves your infrastructure. Not to us, not to anyone.
  • You can inspect every line of code. It's all on GitHub.

Privacy Policy

What we collect

Nothing. Zero. We have no servers running. We have no analytics. We have no way to see your data even if we wanted to.

What the software stores

session-forge hub stores data on your server, in JSON files in a directory you control. This data includes:

  • Agent records (developer names, machine names, API key hashes)
  • Session checkpoints (task descriptions, file paths, status)
  • Decisions and dead ends (technical choices and debugging notes)
  • Journal entries (session summaries)
  • Audit logs (who accessed what API endpoints)

All of this is under your control. You can delete it, back it up, or move it at any time.

External connections

The hub makes zero external network calls by default. The only exceptions:

  • Google Fonts — the dashboard loads the Inter font from Google's CDN. This is a standard web practice but can be self-hosted if needed.
  • Stripe — only if you configure the donate page with Stripe keys. This is entirely optional and disabled by default.

API keys

API keys are bcrypt-hashed before storage. The raw key is shown once at creation time and never stored. If compromised, keys can be rotated without re-registering the agent.

Data retention

You control retention. The hub has configurable entry caps (500 checkpoints, 200 decisions, 100 dead ends, 100 journal entries per agent). You can prune data manually or adjust these limits.

Data Sovereignty

This section matters if you're evaluating session-forge hub for corporate use.

GDPR / CCPA Compliance

Since no data leaves your infrastructure, the standard data processing concerns with third-party SaaS don't apply. You're both the data controller and processor.

SOC 2 / ISO 27001

Self-hosted means the hub falls under your existing security controls and audit scope. No third-party sub-processors to evaluate.

Air-Gapped Environments

session-forge hub works completely offline. No internet required. The dashboard uses no CDN resources (except Google Fonts which can be self-hosted).

Data Residency

Your data lives exactly where you put the server. No cross-border data transfer concerns.

About Donations

session-forge hub is free. Donations are entirely optional and go toward continued development.

If you donate through the donate page:

  • Payments are processed by Stripe — we never see your card details
  • Stripe's standard privacy policy applies to the payment transaction only
  • We receive: the amount, your name (if provided), and email (for receipt only)
  • We do not use donation information for marketing or share it with anyone
